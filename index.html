Cubic Empire

Rust Edition

Проект Cubic Empire Rust Edition (CERE) или Кубическая Империя планируется как инди-игра в жанре песочницы с фрагментами RPG, в первых релизах которой будет сделан акцент на техническую часть: игра будет написана на языке программирования Rust, изначально будет многопоточный и будет иметь открытый API для углубленного моддинга.

Задача организации, в особенности же укрепление и развитие структуры требуют определения и уточнения позиций, занимаемых участниками в отношении поставленных задач
